Overview

    WinTim32 is a fully functional meta assembler and 1.5-pass assembler.

    The full C++ source code is also available for the WinTim32 program, and you are free to modify or distribute this code.

    Check out the WinTIM quick start tutorial   (MS Word version).  Here are some example WinTim source files for a simple computer.
 

Features

    Interface

  • Customizable syntax highlighting for both Meta-Assembly and Assembly files, utilizing user opcodes as they are defined
  • Dynamic tab repositioning for easier readability when editing assembly files
  • Separate dockable output window for error and status messages
  • (LST) output listing
  • Altera Memory Initialization File (MIF) format output object code listing
  • Symbol and definition tables viewable at any time
  • Continuously updating label table
    Assembler
  • Address index by one or by the instruction size for flexibility
  • Memory-based meta-assembly and assembly, requiring no additional files
  • Context-sensitive error messages

Known Bugs
 
     Spelling errors in instruction mnemonics may not always generate an error warning. This can be found by checking the LST file for an instruction that did not generate any machine code in the left column of the listing.
