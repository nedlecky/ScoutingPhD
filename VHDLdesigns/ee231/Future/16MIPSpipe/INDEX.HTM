LC-2 VHDL Synthesis and Simulation Model

 

The LC-2 computer is described in Introduction to Computing Systems from bits & gates to C & Beyond by Yale Patt and Sanjay Patel, McGraw Hill, 2001. This web page contains links to a VHDL model of the LC-2 computer designed to work with the Altera student edition. The LC-2 also can be implemented in hardware on the 70,000 gate student Altera UP 1X board. The control unit uses a VHDL state machine based on the state diagram found in Appendix C. An additional state was added to reset the processor.

The LC-2 requires approximately 35,000 gates and it can be run on an Altera UP 1X board. The two pushbuttons are used for clock and reset. The seven segment displays contain the value of the PC. Additional logic has been added to provide a video display of the registers on an Altera UP 1X board as shown below. A monitor must be plugged into the UP 1X’s VGA connector to see the display.
